Petunjuk dan Database

BUATLAH WEBSERVICE REST API MENGGUNAKAN PHP/NODE JS(PILIH SALAH SATU)
PETUNJUK

totalantrean : adalah Jumlah semua antrean


sisaantrean : adalah jumlah antrean yang belum di panggil : Field (statusdipanggil) 0=belum 1=sudah


DATABASE

CREATE TABLE `antriansoal` ( `nomorantrean` VARCHAR(50) NULL DEFAULT NULL COLLATE 'latin1_swedish_ci', `angkaantrean` INT(11) NULL DEFAULT NULL, `norm` VARCHAR(50) NULL DEFAULT NULL COLLATE 'latin1_swedish_ci', `namapoli` VARCHAR(50) NULL DEFAULT NULL COLLATE 'latin1_swedish_ci', `kodepoli` VARCHAR(50) NULL DEFAULT NULL COLLATE 'latin1_swedish_ci', `tglpriksa` DATE NULL DEFAULT NULL, `nomorkartu` VARCHAR(50) NOT NULL COLLATE 'latin1_swedish_ci', `nik` VARCHAR(50) NOT NULL COLLATE 'latin1_swedish_ci', `keluhan` VARCHAR(50) NOT NULL COLLATE 'latin1_swedish_ci', `statusdipanggil` INT(11) NOT NULL DEFAULT '0', `int` INT(11) NOT NULL AUTO_INCREMENT, PRIMARY KEY (`nomorkartu`) USING BTREE, INDEX `int` (`int`) USING BTREE ) COLLATE='latin1_swedish_ci' ENGINE=InnoDB AUTO_INCREMENT=695 ;


HASIL TES DI KOMPRES DI KIRIM KE HRD

Token

URL : {BASE_URL}/auth

Fungsi : Membuat token

Method : GET

Format : Json

Header :
        x-username: {user akses}
        x-password: {password akses}

                                
{
    "response": {
        "token": "1231242353534645645"
    },
    "metadata": {
        "message": "Ok",
        "code": 200
    }
}

Keterangan:

Contoh generate token dapat menggunakan link berikut [jwt] : https://jwt.io/

Status Antrean

URL : {BASE_URL}/antrean/status/{kode_poli}/{tanggalperiksa}

Contoh URL : {BASE_URL}/antrean/status/001/2020-01-28

Fungsi : Menampilkan status antrean per poli

Method : GET

Format : Json

Header :
        x-token: {token}
        x-username: {user akses}

  • Response

Response

                                            
{
    "response": {
            "namapoli" : "Poli Umum",
            "totalantrean" : "25",
            "sisaantrean" : "4",
            "antreanpanggil" : "A21",
            "keterangan" : ""
    },
    "metadata": {
        "message": "Ok",
        "code": 200
    }
}

Catatan:

Metadata code:
200: Sukses
201: Gagal
Selain metadata code 200, agar message pada metadata diisi sesuai dengan kondisi di lapangan

Ambil Antrean

URL : {BASE_URL}/antrean

Fungsi : Mengambil antrean

Method : POST

Format : Json

Header :
        x-token: {token}
        x-username: {user akses}

  • Request
  • Response

Request

                                            
{
    "nomorkartu": "00012345678",
    "nik": "3212345678987654",
    "kodepoli": "001",
    "tanggalperiksa": "2020-01-28",
    "keluhan": "sakit kepala"
}

Response

                                            
{
    "response": {
            "nomorantrean" : "A12",
            "angkaantrean" : "12",
            "namapoli" : "Poli Umum",
            "sisaantrean" : "4",
            "antreanpanggil" : "A8",
            "keterangan" : "Apabila antrean terlewat harap mengambil antrean kembali."
    },
    "metadata": {
        "message": "Ok",
        "code": 200
    }
}

Catatan:

Metadata code:
200: Sukses
201: Gagal
202: Pasien Baru

Selain metadata code 200, agar message pada metadata diisi sesuai dengan kondisi di lapangan
